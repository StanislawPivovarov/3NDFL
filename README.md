# Порядок запуска проекта:
## 1. Клонировать проект к себе
```
git clone https://github.com/StanislawPivovarov/3NDFL.git
```
## 2. Установить зависимости
```
yarn install
```
## 3. Сбилдить проект
```
gulp build
```
## 4. Запустить проект
```
gulp server
```
